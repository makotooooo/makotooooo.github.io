new 与 Object.create()的区别

这样创建出来的 o1 与 o2 有什么区别呢？

可以看到 o1 继承了构造函数 a 中的属性，而 o2 没有。

new

当调用 new 关键字的时候 js 内部其实做了四件事

  1. 创建一个空对象
  2. 将这个对象的原型指向构造函数的原型
  3. 执行构造函数，绑定 this
  4. return 这个对象

伪代码

Object.create()

Object.create()方法创建一个新对象，使用现有的对象来提供新创建的对象的__proto__。

语法

参数

  • proto

    新创建对象的原型对象。

  • propertiesObject

    可选。如果没有指定为 undefined，则是要添加到新创建对象的不可枚举（默认）属性（即其自身定义的属性，而不是其原型链上的枚举属性）对象的属性描述符以及相应的属性名称。这些属性对应Object.defineProperties()的第二个参数。

返回值

一个新对象，带着指定的原型对象和属性。

其实上面这堆内容翻译过来也就是说创建一个新对象，然后将新对象的__proto__指向传入的对象。
